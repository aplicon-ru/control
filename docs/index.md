# Univerkon Control

Единая операционная консоль экосистемы Универкон — деплой, лицензии,
мониторинг и брендинг всех -конов через один инструмент.

```
docker run -p 8080:8080 -v ./data:/data -v ./keys:/keys:ro ghcr.io/aplicon-ru/control
```
